## Scope

This direction studies how to design foundation model systems that remain stable and useful across heterogeneous tasks.

## Current Questions

- How should routing and memory modules be co-trained?
- What abstractions reduce coupling between reasoning and retrieval?
- Which scaling strategies preserve interpretability?

## Milestones

1. Build a reproducible baseline stack.
2. Benchmark modular variants under distribution shift.
3. Publish system-level analysis and open-source tooling.
